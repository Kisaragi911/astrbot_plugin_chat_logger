# Chat Logger — 私聊对话旁路存档器

AstrBot 插件。以纯旁路方式拦截私聊场景下用户与 LLM 的完整对话，异步写入本地 TXT 文件，**不干涉任何原生消息流程**。

---

## 功能

- 自动记录私聊中用户发送的每条消息
- 自动记录 LLM 的最终文本回复（跳过工具调用响应和流式分块）
- 按 `{用户ID}_{平台名}.txt` 分文件存储，存放在插件目录下的 `logs/` 中
- 30 秒去重窗口：同一用户同一平台同一消息内容在窗口内只记一条
- 全部异步 I/O，不影响事件循环
- 写入失败静默忽略，绝不因日志错误影响 AstrBot 正常运行

## 日志格式

```
[2026-06-06 14:30:01] [user_id_123]: 今天天气怎么样
[2026-06-06 14:30:05] [Bot]: 今天是晴天，气温 25 度左右。
```

## 安装

将插件文件夹放入 AstrBot 的 `data/plugins` 目录即可。AstrBot 会自动加载。

## 注意事项

- 仅记录**私聊**消息，群聊不处理
- 日志文件直接以明文存储，请注意数据安全
- 不提供日志查看 / 导出界面，日志即文件，自行取用

## 配置

本插件无需额外配置，加载即用。
